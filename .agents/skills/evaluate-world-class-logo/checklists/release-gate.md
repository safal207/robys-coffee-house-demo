# Logo release gate

A logo review may recommend release only when all applicable checks pass:

- [ ] exact source asset and revision are recorded;
- [ ] identity, asset, implementation and system defects are separated;
- [ ] no P0 finding remains;
- [ ] primary mark is recognizable at 16 px and 32 px;
- [ ] avatar variant survives a circular crop;
- [ ] mobile and desktop lockups preserve aspect ratio and clear space;
- [ ] one-color and reverse versions remain recognizable;
- [ ] wordmark kerning, apostrophe and diacritics are reviewed;
- [ ] storefront, cup, stamp and low-quality print contexts are considered;
- [ ] SVG has a clean viewBox and no accidental raster/font dependency;
- [ ] public asset revision and Service Worker delivery are current;
- [ ] visual evidence is bound to the exact asset or CSS blob;
- [ ] benchmark references are used as principles, not copied forms;
- [ ] cultural and confusing-similarity risks are documented;
- [ ] trademark clearance, memory testing and business impact are not falsely claimed.
